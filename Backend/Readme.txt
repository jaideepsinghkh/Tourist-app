Hello friends, this folder contains all the files pertaining to the backend of the project. It contains 2 files, namely :-

1.) backend_ERD.png     - png file showing the ERD diagram of the backend of the application.
2.) backend_ERD(HD).png - Higher resolution image of the same.
3.) backend_ERD.erdplus - Software generated job for the backend of the application.

To open the backend_ERD.erdplus, open this link in your web browser : https://erdplus.com/#/standalone

This is the standalone version of the software used to build this ERD.

To optn the file :-

Click on Diagram --> Select Open Diagram File --> Choose backend_ERD.erdplus in the file upload dialog box --> Click Open.
